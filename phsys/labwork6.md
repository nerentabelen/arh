# Лабораторная работа № 6

## Определение основных характеристик и тестирование видеосистемы ПК

**Тема:** Видеосистема персонального компьютера

**Цель работы:** Изучить современные видеокарты на графических процессорах NVIDIA, AMD (ATI) и технологии объединения видеокарт

**Время выполнения:** 1 час

**Оборудование:** учебный персональный компьютер

**Программное обеспечение:** операционная система, презентация, тестовые программы

---

## Теоретические основы

Для выполнения лабораторной работы использовались две утилиты:
- **GPU-Z v0.6.0** — вывод информации о графическом адаптере
- **FurMark v1.10.2** — тестирование производительности видеокарты

**GPU-Z** поддерживает видеокарты NVIDIA и ATI. Программа позволяет получить следующую информацию:
- модель видеокарты и интерфейс подключения
- характеристики графического процессора (версия BIOS, ревизия чипа, частоты в 2D/3D-режимах)
- поддержка DirectX
- параметры видеопамяти (тип, объём, разрядность шины)

## Анализ результатов

### Характеристики адаптера

| Параметр | Оценка |
|----------|--------|
| Тип GPU | Intel(R) HD Graphics — интегрированная графика |
| Техпроцесс | 22 нм (поколение 2013–2015 гг.) |
| DirectX | 11.0 — поддержка базовых API |
| Память | DDR3 (системная, разделяемая) |
| Частоты | 650 МГц (база) → 1100 МГц (под нагрузкой) |
| Драйвер | 10.18.10.4252 — версия 2015–2016 гг. |

### Положительные аспекты

- **Термостабильность:** температура под нагрузкой не превышает 45°C
- **Стабильность работы:** частоты GPU поднимаются до 1100 МГц, тесты пройдены без сбоев
- **Энергоэффективность:** интегрированное решение с низким TDP

### Ограничения

- **Низкая 3D-производительность:** 2–3 FPS в стресс-тестах
- **Устаревший драйвер:** отсутствуют оптимизации для нового ПО
- **Отсутствие выделенной видеопамяти:** использование системной DDR3 снижает пропускную способность
- **Ограниченная поддержка API:** отсутствует поддержка DX12 Ultimate и трассировки лучей

### Оценка пригодности

| Задача | Пригодность |
|--------|-------------|
| Офисные задачи / веб-сёрфинг | Подходит |
| Просмотр видео (до 1080p) | Подходит |
| Лёгкие/старые игры | Только на низких настройках |
| Современные игры / монтаж / 3D | Не подходит |
| Техническое состояние | Исправен |

## Контрольные вопросы

### 1. Какие программы используют для тестирования видеокарт?

| Программа | Назначение |
|-----------|-----------|
| FurMark | Стресс-тестирование, проверка стабильности охлаждения |
| MSI Afterburner | Разгон и мониторинг параметров GPU |
| 3DMark | Игровой бенчмаркинг |
| GPU-Z | Информация о видеокарте, мониторинг сенсоров |
| AIDA64 | Комплексная диагностика системы |
| Unigine Superposition | Графический бенчмарк |
| OCCT | Тестирование стабильности, проверка на артефакты |
| Geekbench / Basemark GPU | Вычислительные тесты |

### 2. Какие программы используют для тестирования мониторов?

| Программа | Платформа | Функции |
|-----------|-----------|---------|
| PassMark MonitorTest | Windows | 35 тестовых паттернов, проверка пикселей, цвета, гаммы |
| LCD Monitor Test | Windows | Поиск дефектных пикселей, проверка подсветки |
| Display Tester | Android | Тесты для LCD/OLED-экранов |
| EIZO Monitor Test | Веб-версия | Геометрия, время отклика, цветопередача |
| TestUFO | Веб-версия | Частота обновления, motion blur, tearing |

**Проверяемые параметры мониторов:**
- битые/залипшие пиксели
- контраст и яркость
- цветопередача и калибровка
- геометрия и резкость
- время отклика
- равномерность подсветки

## Вывод

Видеоадаптер Intel(R) HD Graphics находится в исправном техническом состоянии: система охлаждения работает эффективно (температура под нагрузкой не превышает 45°C), частоты соответствуют заявленным, тесты пройдены без сбоев. Однако производительность адаптера недостаточна для современных игр и ресурсоёмких задач (2–3 FPS в стресс-тестах). Данное интегрированное решение пригодно для офисных задач, веб-сёрфинга и воспроизведения видео, но для работы с графикой, монтажа или игр требуется установка дискретной видеокарты.
